'this' keyword

  • in global scope : 'this' keyword points to global window object. 
        console.log(this)
  • in regular function call 'this' keyword is global window object. and in strict mode'use strict' this keyword is undefined
  • in method call means function attach to obj. 'this' keyword points to object on which that method is called.
  • arrow function dosen't have it's own 'this' keyword.. it uses this keyword of it's parent scope.
